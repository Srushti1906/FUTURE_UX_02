# Appointment Booking App UI

## Project Overview

This project is a mobile-first UI design for an appointment booking application created as part of a UI/UX internship task.

The goal is to design a simple and intuitive mobile interface that allows users to easily book appointments with service providers.

## Business Example

Glow & Style Salon – Pune

## Target Users

* People who want to book salon appointments quickly
* Busy professionals
* Students
* Customers who prefer online booking instead of phone calls

## Booking Flow

1. User logs in with phone number
2. Selects a service
3. Chooses a stylist
4. Selects date and time
5. Confirms the appointment

## UX Decisions

* Clean and minimal interface
* Large touch-friendly buttons
* Clear navigation between steps
* Simple booking flow for first-time users

## Screens Designed

* Login Screen
* Service Selection Screen
* Staff Selection Screen
* Date & Time Slot Selection
* Appointment Confirmation Screen
